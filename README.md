# Perfume Finder  ароматизатор

Este es un simple proyecto web que te ayuda a descubrir tu perfume ideal basado en las notas olfativas que más te gusten.

## ¿Cómo funciona?

La aplicación te presenta una selección de notas olfativas (como vainilla, jazmín, cítricos, etc.). Simplemente selecciona cuatro de estas notas y la aplicación te sugerirá un perfume que contenga una combinación de tus elecciones.

Para usar la aplicación:
1.  Abre el archivo `PERFUMES/index.html` en tu navegador web.
2.  Haz clic en cuatro notas que te gusten.
3.  Una vez que hayas seleccionado cuatro notas, haz clic en el frasco de perfume.
4.  La aplicación te mostrará un perfume que podría gustarte.
5.  Puedes hacer clic en "Jugar otra vez" para empezar de nuevo.

## Estructura del repositorio

*   `PERFUMES/index.html`: La página principal de la aplicación.
*   `PERFUMES/elegituperfume.html`: Un duplicado de `index.html`.
*   `PERFUMES/*.jpeg`, `PERFUMES/*.jpg`: Imágenes de los frascos de perfume.
*   `*.jpeg`, `*.jpg`: Imágenes de las notas olfativas.

## Tecnologías utilizadas

Este proyecto está construido con:
*   HTML5
*   CSS3
*   JavaScript

Es un proyecto puramente de frontend, por lo que no requiere de un backend para funcionar.
